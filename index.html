Git Commands (new branch)

$ git init

Initialize Local Git Repository

$ git add (file)

Add File(s) to Index

git add *.html - add every .html file

git add . - add every file

$ git rm --cached (file)

Remove File(s) from Index

$ git status

Check Status Of Working Tree

$ git commit

Commit Changes In Index

git commit -m '(comment)'

$ git push

Push To Remote Repo

$ git pull

Pull Latest From Remote Repo

$ git clone

Clone Repo Into a New Directory

$ touch .gitignore

Creates .gitignore file, you can write into it filename you dont want to commit (for example logs)

$ git branch (branch name)

Creates new branch

$ git chekout (branch name)

Switch branch
